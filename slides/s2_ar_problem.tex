  \frametitle{Ackroyd-Riyait Problem}
  % describes the Ackroyd-Riyait problem
  The problem of interest is the \textit{Ackroyd-Riyait problem}, which is a non-multiplying 2D neutron transport model that was created to demonstrate a numerical artifact called \textit{ray effects}~\cite{ACKROYD19891}.
  These appear when solving low-scattering (non-diffusive) problems using discrete ordinates methods.
